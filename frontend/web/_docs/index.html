Installation Guide

First of all, copy code folder and paste to your server/localhost.

  • Download Zip file From Codecaniyon
  • Extract Zip into your root directory or public_html folder in your server.
  • DataBase Setting
    • Step 1: Open PhpMyAdmin panel
    • Step 2: Create new database.
    • Step 3: Click on import link for import SQL File
    • Step 4: Select HouseVilla.SQL file and go
    • DataBase Successfully Created
    • Step 5: Edit Config file:
      • Step 1: goto HouseVilla/common/config/main-local.php
      • Step 2: Change Host name
      • Step 2: Change User name
      • Step 2: Change Password
      • Step 2: Change DataBase name
  • .htaccess setting we use two .htaccess file in the script
    First: HouseVilla/frontend/web/.htaccess
    second: HouseVilla/.htaccess
    make change only in second .htaccess file according to your requirement
  • All is done Installation completed. click on frontend and enjoy with Swag
